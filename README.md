# Course Picker Home (Static HTML/CSS)

This is a phone-sized, static HTML/CSS implementation of **Screen 1** of a high-fidelity prototype:
**Course Picker Home**.

## How to Run

This is a simple static HTML/CSS project that requires no build process or server.

### Method 1: Double-Click (Easiest)
1. Navigate to the project folder in Finder
2. Double-click `index.html`
3. The page will open in your default web browser

### Method 2: From Terminal
```bash
cd "/your-folder-location"
open index.html
```

### Method 3: Drag and Drop
1. Open Chrome (or your preferred browser)
2. Drag the `index.html` file into the browser window

## What to Expect

This prototype displays a phone-sized course picker interface with:
- Header with search functionality (visual only)
- Filter chips and tabs
- Three course cards (STAT 1020, CSCI 2470, PSYC 001)
- "At a Glance" calendar view
- Bottom navigation bar

## Testing the UI

### Recommended: Chrome DevTools
1. Open the page in Chrome
2. Press `Cmd+Option+I` (Mac) or `F12` (Windows/Linux) to open DevTools
3. Press `Cmd+Shift+M` (Mac) or `Ctrl+Shift+M` (Windows/Linux) to toggle device toolbar
4. Test at different phone screen sizes (iPhone 12, Pixel 5, etc.)
